## 新学单词

```js
umbrella // 伞,雨伞
please // 请
here // 在这里,此时
my // 我的
ticket // 票,入场券
number // 数字,编号
five // 五,五个
sorry // 抱歉,惭愧的,愧疚的
sir // 先生,长官
madam // 夫人,女士
cloakroom // 衣帽间,衣帽存放处
```

## 表达拜托办某一件事

```js
// 比如一个小孩想求家长买一个玩具
小孩: please please please // 拜托
```

## here延伸

```js
here // 在这里,此时

come here // 过来,来这里
jack is here // jack在这里
is jack here? // jack在这里吗?
```

## ticket延伸

```js
ticket // 票,入场券

one way ticket // 单程票
return ticket // 往返票
round-trip ticket // 往返票

// 如何像售票员表达你想买一张去北京的单程票或往返票?
one ticket to BeiJing. // 大部分情况下会默认理解为单程票
one way ticket to BeiJing. // 明确表达买一张去北京的单程票

one return ticket to BeiJing. // 一张到北京的往返票
```

## number延伸

```js
number // 数字,编号

one two three four five six seven eight nine ten // 1 ~ 10

// 如何用numberl来表示任何一个数字,比如用number来表示115和150.
one one five // 115
one five zero // 150

// 表示某一个电话号码 比如 1555289249
one five five five two eight nine two four nine
```

## sorry延伸

```js
sorry // 抱歉,惭愧的,愧疚的

// 当你不小心撞了某人一下
sorry 

// 当你做了一件较为严重错误的事
i am so sorry
```

## sir与madam延伸

```js
sir // 先生,长官
madam // 夫人,女士

// 当你在一个正式场合中想表示对一个男性的感谢时
thanks sir //  谢谢先生
thank you sir // 谢谢你先生
thank you madam // 谢谢你女士

// 当你在一个不是正式场合中想表示对一个男性的感谢时
thanks man // 谢谢你,伙计
```



## 文章对话

```js
A: My caot and my umbella plaese // 请把我的外套和雨伞拿给我
A: Here is my ticket // 这是我的票
A: thank you sir // 谢谢你,先生
A: Number five // 五号
B: Here's your umbrella and your coat. // 这是你的雨伞和你的外套
A: This is not my umbrella sorry sir. // 抱歉先生,这不是我的雨伞
B: Is this your umbrella? // 这个是你的雨伞吗?
A: No, itsn't // 不,它不是
B: Is this it? // 这个是吗?
A: Yes, it is. // 是的,这是
A: Thank you very much. // 非常感谢
```

## 把某些东西递给人时

```js
Here is my ticket // 这是我的票
Here is my money // 这是我的钱
Here is my cellphone // 这是我的手机
Here is my coat // 这是我的外套
take my coat // 给你我的外套
```
