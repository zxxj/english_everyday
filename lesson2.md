## 新学单词

```js
pen // 笔,钢笔
penceil // 铅笔
book // 书
watch // 手表
coat // 外套,大衣
dress // 连衣裙
skirt // 裙子
shirt // 衬衫
car // 车
house // 房子
```

## 两种夸人方式?

```js
// 正常直接夸赞对方衣服好看
A: nice coat man!
B: thank you man.

// 比较夸张的夸赞对方衣服或物品好看
I love your skirt
I love your watch
```
